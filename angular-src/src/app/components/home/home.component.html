Excel Keeper

Welcome to our custom MEAN-based app used for uploading, processing and storing Excel files.

Register Login

Express Backend

A Node.js/Express server with the help of Mongoose is used to organize models and query the database. Files are checked, uploaded and kept a MongoDB database.

Angular 2 Frontend

Angular-CLI was used to generate components, services and more. The perfect base for a Single-Page Application.

JWT Tokens

Full featured authentification using JSON web tokens. Each user has access only to his or her files.
